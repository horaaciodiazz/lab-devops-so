# Laboratorio N°6: DevOps Moderno con GitHub Actions y Azure DevOps

Repositorio del laboratorio de Sistemas Operativos (UCU).

## Estructura

- app/       API dummy en Node y Express, con endpoint /health
- pipeline/  scripts de apoyo para el despliegue (se completa en la Parte 4)

## Correr localmente

  cd app
  npm install
  npm start

El servidor queda en http://localhost:3000
